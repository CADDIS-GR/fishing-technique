🛠️ 404 에러 해결 체크리스트

만약 링크 접속 시 404 에러가 발생한다면 아래 사항을 확인해 보세요:

배포 대기 시간: GitHub Pages 설정 후 실제 배포까지 약 1~3분 정도 시간이 소요됩니다.

Settings 확인: 저장소의 Settings > Pages 메뉴에서 Branch가 main (또는 master)으로 선택되어 있고 / (root) 폴더가 설정되어 있는지 확인하세요.

파일 위치: index.html 파일이 저장소의 **최상위 경로(Root)**에 있는지 확인하세요. 폴더 안에 들어가 있으면 안 됩니다.

대소문자 구분: 깃허브 서버는 대소문자를 엄격히 구분합니다. index.html이 혹시 Index.html로 되어 있지 않은지 확인해 보세요.

폴더 이름: index.html 내의 링크 경로(techniques/...)와 실제 폴더 이름이 정확히 일치해야 합니다.
